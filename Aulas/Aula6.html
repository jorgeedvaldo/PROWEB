Selector Atributo

Aplica-se os estilos nos elementos através dos seus atributos. Exemplo:

strong[tittle=info]{
color: red;
font.size: 2em;
}

Se no seu titulo tiver duas palavras, ou mais, separadas por espaço, podemos usar o til (~) para estilizar o elemento apartir de umas das palavras

<strong tittle="info ista"> Bla, bla. </strong> strong[tittle ~=info]{
color: red;
font.size: 2em;
}
